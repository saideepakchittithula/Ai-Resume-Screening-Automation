# AI Resume Screening & Interview Assessment Automation

## Overview
This project automates the recruitment process using n8n and AI.

The workflow analyzes candidate resumes, generates interview questions, evaluates responses, calculates scores, and sends automated email notifications.

## Features

- Resume collection through forms
- AI-powered resume analysis
- Automatic interview question generation
- Candidate answer evaluation
- Automated score calculation
- Eligibility filtering
- Email notifications
- Google Sheets integration
- Recruitment data management

- n8n
- OpenAI
- Google Sheets
- Gmail
- JavaScript

## Workflow

1. Candidate submits resume
2. AI analyzes resume
3. AI generates 5 interview questions
4. Candidate submits answers
5. AI evaluates responses
6. Score is calculated automatically
7. Eligible candidates receive selection emails
8. Data is stored in Google Sheets
